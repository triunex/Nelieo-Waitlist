# AI OS Screenshots

Place your Nelieo AI OS screenshots here.

## Naming Convention:
- screenshot-1.png
- screenshot-2.png
- screenshot-3.png
- screenshot-4.png
- screenshot-5.png
- screenshot-6.png

## Recommended Settings:
- **Format**: PNG or JPG
- **Resolution**: 1920x1080px (Full HD)
- **File Size**: Under 2MB each for fast loading

## Tips:
- Capture clean, professional screenshots showing key features
- Include workflows, agent execution, dashboard views
- Show different aspects: planning, execution, results
- Make sure text is readable
- Avoid sensitive data

The images will automatically appear in the gallery on `/insights.html`
